# Bank Management System
## Description
A console-based Bank Management System built using Java, JDBC, and MySQL.
## Features
- Create Bank Account
- View All Accounts
- Search Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Check Balance
- Update Account Details
- Delete Account
- View Transaction History
- MySQL Database Integration
- Exception Handling
- Menu-Driven Interface
  ## Technologies Used
  - Java
  - JDBC
  - MySQL
  - Maven
  - Intellij IDEA
 ## Database
 MySQL is used to store and manage bank account information.
